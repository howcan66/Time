10 BASIC WEB COMPONENT BUILDING BLOCKS (Reference Guide)
================================================================================

AI TD1 - TEXT DISPLAY
   - Use <p> for paragraphs, <h1>-<h6> for headings
   - Use <div> to group related content
   - Style with CSS: color, font-size, text-align

AI TD2 - TEXT INPUT FORMS
   - Use <input type="text"> for single-line input
   - Use <textarea> for multi-line input
   - Add <label> to describe each input
   - Wrap in <form> for organization

AI TD3 - RADIO BUTTONS
   - Use <input type="radio"> for single choice from multiple options
   - Group related radios with same "name" attribute
   - Use <label> to make clickable text

AI TD4 - DROPDOWN/SELECT LISTS
   - Use <select> and <option> tags
   - Let users pick one from many options
   - Style with CSS for consistency

AI TD5 - BUTTONS & INTERACTIONS
   - Use <button> or <input type="button">
   - Add onclick attribute to trigger JavaScript functions
   - Style with CSS for appearance

AI TD6 - LAYOUT ROWS & COLUMNS
   - Use <div> with CSS Flexbox or Grid
   - Flexbox: display: flex; for horizontal rows
   - Grid: display: grid; for complex layouts
   - Use CSS justify-content and align-items

AI TD7 - SCROLLBARS & OVERFLOW
   - Use overflow: scroll; for scrollable content
   - overflow: hidden; to hide excess
   - overflow: auto; for automatic scrollbars
   - Set max-height on <div> to enable scrolling

AI TD8 - FRAMES & CONTAINERS
   - Use <div> with borders and padding
   - Add border: 1px solid #ccc;
   - Use padding/margin for spacing
   - Background color: background-color: #f0f0f0;

AI TD9 - DESIGN & STYLING BASICS
   - Use CSS in <style> tag or external CSS file
   - Define colors, fonts, sizes consistently
   - Use classes for reusable styles: <div class="container">
   - Mobile-friendly: use max-width, padding, responsive units

AI TD10 - COMBINING ELEMENTS INTO PAGES
    - Structure: <header>, <main>, <footer>
    - Group related elements in <section> or <div>
    - Use consistent spacing and alignment
    - Test on different screen sizes (mobile, tablet, desktop)
